# Global Strategic Tech - Brand Guidelines & Visual Identity System
**Version 2.0 | January 2026**

---

## Executive Summary

This brand guide documents the existing visual identity of Global Strategic Tech (GST) as deployed on globalstrategic.tech. The brand centers on executive-level credibility, technical precision, and measurable business transformation, with the Greek Delta (Δ) symbol representing change and transformation.

---

## 1. Logo & Brand Mark System

### Primary Brand Mark: GST with Delta Symbol

**Current Implementation:**
The brand uses "GST" as a simple, clean text mark accompanied by the Delta symbol (Δ) which appears prominently in the footer and represents transformation and change - core to the value proposition.

**Delta Symbol Significance:**
- **Mathematical Context:** Represents change, difference, variation
- **Business Context:** Transformation, improvement, value creation
- **Brand Context:** The measurable impact GST delivers to portfolio companies

### Logo Variations

**1. Primary Horizontal Lockup**
```
GST Δ  Global Strategic Tech
```
- Clean sans-serif typography
- Delta symbol integrated as visual punctuation
- Used in: Header navigation, email signatures, primary materials

**2. Text-Only Logo**
```
GST
```
- Minimal, confident three-letter abbreviation
- Used in: Navigation, compact spaces, mobile header

**3. Delta Symbol as Icon**
```
Δ
```
- Standalone icon for favicons, social media, watermarks
- Geometric, recognizable, scalable
- Represents brand essence: transformation

**4. Full Name**
```
Global Strategic Tech
```
- Used in: Formal documents, contracts, footer

### Logo with Delta - Design Specifications

**Recommended Lockup:**
```
   Δ GST
```
or
```
GST Δ
```

**Typography:**
- Clean, geometric sans-serif
- Medium to bold weight for confidence
- Tight letter-spacing for technical precision
- Delta symbol same size or slightly larger than text

**Clear Space:**
- Minimum clear space: Equal to height of Delta symbol on all sides
- Never crowd other elements around the Delta

**Minimum Sizes:**
- Digital: 120px width minimum
- Print: 1 inch width minimum
- Favicon: 32px Delta symbol only

## 3. Typography System

**Sans-Serif System:**
The site currently uses a clean, modern sans-serif font stack, likely:
- System UI fonts for performance and clarity
- Fallback hierarchy: -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif

**Characteristics Observed:**
- Clean geometric letterforms

- Regular weight for body copy
- Excellent readability at all sizes
- Professional without being generic

- Bold weight
- Primary dark color
- Generous line height

- Standard reading size (16-18px)
- Regular weight
- Medium gray for comfortable reading
- Good line height (1.5-1.7)

- Small to medium (14-16px)
- Medium weight
- Compact letter-spacing
- All caps or mixed case

**Footer/Legal:**
"© 2026 Global Strategic Technologies"
- Small size (12-14px)
- Regular weight
- Muted gray color

### Monospace/Data Font

For technical specifications and metrics:
- Monospace font family
- Used for: ARR figures, dates, technical specs, data tables
- Maintains precision and scannability

## 6. Delta Symbol (Δ) - Brand Integration Guide

### Why Delta?

**Mathematical Significance:**
- Represents change: ΔX (change in X)
- Represents difference: Δ between before and after
- Represents improvement: Δ EBITDA, Δ Revenue

**Brand Alignment:**
- GST delivers measurable transformation
- Focus on value creation and change
- Technical precision with business impact
- Before/after business metrics

### Delta Usage Guidelines

**Primary Uses:**
1. **Logo Integration** - Part of main brand mark
2. **Footer Signature** - End of page brand moment
3. **Metric Presentation** - Δ$8.2M EBITDA
4. **Section Dividers** - Visual separator with meaning
5. **Favicon** - Simple, recognizable icon

**Typography Considerations:**
```css
Delta Symbol: Δ
Unicode: U+0394
HTML: &Delta; or &#916;
CSS: content: '\0394';
```

**Size Relationships:**
- Logo: Delta same height as capital letters (G, S, T)
- Metrics: Delta slightly smaller than numbers
- Decorative: Delta can be larger for emphasis

**Color Applications:**
- Matches text color in most contexts
- Can use accent color for emphasis
- Maintains contrast for readability

### Delta-Based Design Elements

**Metric Cards:**
```
    Δ $8.2M
    EBITDA Impact
```

**Section Headers:**
```

Value Creation Portfolio
```

**Before/After Comparisons:**

**Case Study Markers:**

## 10. Logo Design Specifications - Delta Integration

### Recommended Logo Marks (Using Delta)

**Option 1: Delta Prefix**
```
Δ GST
```
- Delta leads, establishing transformation focus
- Clean, geometric alignment
- Easy to scale and reproduce

**Option 2: Delta Suffix**
```
GST Δ
```
- Delta as closing mark
- Signature-style placement
- Punctuation of transformation

**Option 3: Delta Integration**
```
G Δ ST
```
- Delta replaces the triangle shape in "A"
- Integrated into letter forms
- More complex but distinctive

**Option 4: Vertical Stack**
```
  Δ
 GST
```
- Delta above GST
- Symbolic hierarchy
- Square format

**Option 5: Delta + Full Name**
```
Δ Global Strategic Tech
```
- Used for formal applications
- Full brand name with symbol
- Maximum clarity

### Construction Grid for Delta Logo

**Proportions:**
```
Delta Height: 100%
GST Height: 90-100% of Delta
Spacing: 0.25-0.5x Delta width
```

**Alignment:**
- Delta and GST centered on same baseline
- Or Delta sits slightly above baseline
- Optical centering over mathematical centering

**Weight Balance:**
- Delta stroke weight matches GST letter stroke
- Or Delta slightly lighter for elegance
- Maintain visual balance
