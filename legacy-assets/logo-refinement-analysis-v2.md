## üîß Technical Implementation Notes

### File Naming Convention
Use this system for organization:

```
gst-logo-[name]-[layout]-[color]-[bg].[format]

Examples:
gst-logo-full-horizontal-color-transparent.svg
gst-logo-gstech-horizontal-black-transparent.png
gst-logo-gst-icon-color-transparent.svg
```

### Export Specifications

**For Web:**
- Format: SVG (preferred) or PNG
- Background: Transparent
- Size: 2048px wide max (horizontal)
- Optimization: Run through SVGO or TinyPNG

**For Print:**
- Format: PDF or AI (vector)
- Color Mode: CMYK
- Resolution: 300 DPI minimum
- Bleed: Add 0.125" if needed

**For Social:**
- Format: PNG or JPG
- Dimensions: Platform-specific (see design brief)
- Compression: Optimize for file size
- Profile: sRGB color space
